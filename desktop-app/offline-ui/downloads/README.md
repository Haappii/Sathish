## Mobile app download files

The About page can download an Android APK from:

- `VITE_ANDROID_APK_URL` (recommended for production/CDN), or
- default fallback: `/downloads/haappii-billing.apk`

If you want to use the default fallback, place your APK here:

- `shop-billing-app/frontend/public/downloads/haappii-billing.apk`

Then rebuild/redeploy the frontend so the file is served.
