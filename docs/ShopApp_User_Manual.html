Haappii Billing

User Manual: Menus and Navigation

Simple, readable guide for billing staff, managers, and admins to navigate the application quickly.

Download User Manual PDF Download Brochure PDF

1. Start Here

  1. Login and open Home (`/home`).
  2. Use the left sidebar for all navigation.
  3. Menus shown depend on role and permissions.

2. Core Menus

MenuRouteUse
Home/homeDashboard and quick actions.
Billing / Take Away/sales/createCreate bills and checkout flow.
Billing History/sales/historyReview old invoices and edits.
Inventory/inventoryTrack stock and quantities.
Customers/customersCustomer records and due tracking.
Expenses/expensesDaily expense entry.
Reports/reportsSales and business reporting.
Admin / Setup/setupConfiguration and management modules.

3. Setup Section

  1. Categories: `/setup/categories`
  2. Items: `/setup/items`
  3. Shop Details: `/setup/shop`
  4. Users and Permissions: `/setup/users`, `/setup/permissions`
  5. Branches and Suppliers: `/setup/branches`, `/setup/suppliers`
  6. Purchase Orders and utilities: `/setup/purchase-orders`, `/setup/excel-upload`, `/setup/mail-scheduler`

4. Hotel Mode Menus (If Enabled)

  1. Table Billing: `/table-billing`
  2. QR Orders: `/qr-orders`
  3. Order Live and KOT: `/order-live`, `/kot`
  4. Reservations, Delivery, Recipes: `/reservations`, `/delivery`, `/recipes`

5. Daily Navigation Routine

  1. Home check -> alerts, quick stats, reminders.
  2. Billing/Order flow during operations.
  3. Inventory and dues checks mid-day.
  4. Reports and day close at end of shift.
Tip: If a menu is not visible, check the logged-in role and permissions first.
